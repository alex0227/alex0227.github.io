### 命令行修改启动模式
查看现有模式
```
systemctl isolate
```
四种模式emergency.target,graphical.target,multi-user.target,rescue.target

查看当前启动模式
```
systemctl get-default
```
设置启动模式
```
systemctl set-default graphical.target
```
